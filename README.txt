# Atelier-Jaspersoft---Visualize.js

L’intégralité de cet atelier est documenté dans une page HTML.

INSTALLER LA PAGE HTML
> Une fois ce projet téléchargé, décompressé-le et ouvrez le dossier DEMARRER qu'il contient.
> Copiez le dossier *atelier_visualizejs* et collez-le dans le dossier apache-tomcat/webapps du bundle JasperReport Server ou votre serveur applicatif.

OUVRIR LA PAGE
Une fois le dossier copié dans Tomcat, et si Tomcat tourne en local sur le port par default, la page de l’atelier sera disponible en suivant le lien suivant. Sinon, la base *localhost:8080* devra être changé pour pointer vers votre serveur.
http://localhost:8080/atelier_visualizejs/

(Un redémarrage du serveur est facultatif mais recommandé)

NOTE IMPORTANTE
Les fichiers solutions intermédiaires pour chaque module ont été enregistrés dans le dossier SOLUTIONS
